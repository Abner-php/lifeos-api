# 🚀 LifeOS API

API REST desenvolvida com **FastAPI** para gerenciamento de tarefas.

Este projeto permite criar, listar, atualizar, concluir e deletar tarefas, simulando um sistema de produtividade pessoal.

---

## 🧠 Funcionalidades

- ✅ Criar tarefas
- 📋 Listar todas as tarefas
- 🔍 Buscar tarefa por ID
- ✏️ Atualizar tarefa
- ✔️ Marcar tarefa como concluída
- 🗑️ Deletar tarefa

---

## 🛠️ Tecnologias utilizadas

- Python
- FastAPI
- Pydantic
- Uvicorn
- JSON (armazenamento de dados)
